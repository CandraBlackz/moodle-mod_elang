moodle-mod_elang

A moodle module for learning language

View on GitHub Download .zip Download .tar.gz

Description

A moodle module (official moodle page) for learning language (Open Hub page) using the google chrome, firefox, internet explorer or safari browsers. See http://e-lang.github.io/ for other works. See the wiki pages for an help.
